# git-all 🧹

a fast cli tool to initialize, commit, and push all your git projects in a directory at once.

## prerequisites
- `git`
- `gh` (github cli, authenticated)

## setup
clone the repository anywhere on your system:
```bash
git clone [https://github.com/aGuyCalledT/git-all.git](https://github.com/aGuyCalledT/git-all.git) ~/.git-all
